# House-Price-Prediction-Using-Linear-Regression

## Project Overview
This project predicts house prices using the Linear Regression algorithm. It was developed as part of the EduSkills Machine Learning & Data Science Virtual Internship.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization
- Linear Regression model
- Model evaluation
- House price prediction

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Housing Prices Dataset:
https://www.kaggle.com/datasets/yasserh/housing-prices-dataset

## Workflow
1. Load dataset
2. Data preprocessing
3. Exploratory Data Analysis
4. Train-test split
5. Linear Regression model
6. Model evaluation
7. Price prediction

## Results
The model predicts house prices based on housing features and was evaluated using Mean Squared Error (MSE) and R² Score.
